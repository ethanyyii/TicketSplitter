# TicketSplitter 設計方向探索

## 設計目標
建立一個 Apple 官網風格的極簡、現代、高對比 SaaS 首頁，使用 Bento Box 佈局展示三個航空公司的機票分帳功能。

---

## 設計方向 1: 極簡主義 + 高對比 (Minimalism with Bold Contrast)
**設計運動**: Apple 官網風格、Bauhaus 極簡主義

**核心原則**:
- 大量留白，內容稀疏但精準
- 黑白為主，單一強調色
- 超大排版，簡潔有力
- 極少裝飾，功能即美學

**色彩哲學**:
- 背景: 純白 (`#FFFFFF`)
- 文字: 深灰/黑 (`#1a1a1a`)
- 強調色: 深藍 (`#0066FF`) 或 黑色
- 卡片: 淺灰背景 (`#F5F5F5`)，邊框極細

**佈局範式**:
- Bento Box 採用不規則網格 (3 個卡片，可能有不同尺寸)
- 標題區塊佔據全寬，下方卡片左右對齊
- 卡片之間間距寬敞 (32px-48px)

**標誌性元素**:
- 細線邊框 (1px)，無填充
- 極簡圖標 (Lucide React)
- 單色漸層 (黑到透明)

**互動哲學**:
- 最小化動畫，只在必要時出現
- 懸停時背景色微調
- 拖曳時邊框加粗，背景淡化

**動畫設計**:
- 淡入淡出 (200-300ms)
- 縮放 (1.0 → 1.02)
- 邊框漸變 (灰 → 黑)

**排版系統**:
- 標題: Geist Sans (700) 或 System Font 加粗，64px-72px
- 副標題: Geist Sans (400) 或 System Font，20px-24px
- 卡片標題: 600 weight，18px-20px
- 正文: 400 weight，14px-16px

## 設計方向 2: 玻璃態 + 漸層 (Glassmorphism with Gradients)
**設計運動**: 現代 UI 設計、Neumorphism 演進版

**核心原則**:
- 半透明玻璃效果，模糊背景
- 柔和漸層，色彩豐富
- 圓角卡片，陰影深邃
- 光線感與層次感

**色彩哲學**:
- 背景: 深藍漸層 (`#0A1929` → `#132F4C`)
- 卡片: 白色半透明 (rgba 80-90% opacity)
- 強調色: 青藍 (`#00D9FF`) 或 紫色 (`#A78BFA`)
- 文字: 深色 (卡片內) 或 白色 (背景上)

**佈局範式**:
- Bento Box 規則 3 列網格
- 卡片間距均勻 (24px)
- 標題區塊有背景色，與卡片區隔

**標誌性元素**:
- 玻璃效果 (backdrop-filter: blur)
- 彩色漸層邊框
- 發光陰影 (box-shadow with glow)

**互動哲學**:
- 懸停時卡片上浮 (transform: translateY)
- 邊框漸層動畫
- 背景模糊加強

**動畫設計**:
- 上浮 (0 → -8px)，300ms
- 漸層流動 (background-position animation)
- 陰影脈動 (box-shadow 變化)

**排版系統**:
- 標題: Poppins (700) 或 DM Sans (700)，60px-68px
- 副標題: Poppins (500)，18px-22px
- 卡片標題: 600 weight，16px-18px
- 正文: 400 weight，13px-15px

## 設計方向 3: 新銳科技 + 動態 (Tech-Forward with Motion)
**設計運動**: 科技公司風格 (Stripe, Vercel)、動態設計

**核心原則**:
- 豐富的微互動與動畫
- 漸層與色彩層次
- 現代幾何圖形
- 強調流動與變化

**色彩哲學**:
- 背景: 深灰 (`#0F172A`)
- 卡片: 深色背景 (`#1E293B`)，彩色邊框
- 強調色: 多色系 (紅 `#FF6B6B`, 綠 `#51CF66`, 藍 `#339AF0`)
- 文字: 白色或淺灰

**佈局範式**:
- Bento Box 不規則網格 (卡片大小不同)
- 第一個卡片較大 (2x1)，其他 1x1
- 卡片間距 16px-20px

**標誌性元素**:
- 彩色邊框，每個卡片不同顏色
- 動態背景圖案 (SVG 動畫)
- 發光效果 (glow on hover)

**互動哲學**:
- 懸停時邊框發光
- 拖曳時卡片旋轉微角度
- 進度條動畫

**動畫設計**:
- 邊框漸變 (1px → 2px)，300ms
- 旋轉 (0deg → 2deg)，300ms
- 發光脈動 (box-shadow 循環)

**排版系統**:
- 標題: Sora (700) 或 Outfit (700)，56px-64px
- 副標題: Sora (500)，16px-20px
- 卡片標題: 600 weight，14px-16px
- 正文: 400 weight，12px-14px

---

## 最終選擇: 方向 1 (極簡主義 + 高對比)

選擇理由:
- 最符合 Apple 官網風格
- 高對比設計易於閱讀，專業感強
- 拖曳上傳互動在極簡背景下更突出
- 加載狀態與動畫在留白中更明顯

此設計將確保 TicketSplitter 呈現高端、可信、專業的品牌形象。
